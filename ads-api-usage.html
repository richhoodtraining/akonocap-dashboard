Akonocap Dashboard – Facebook Ads API Usage

Akonocap Dashboard connects a user’s Facebook ad accounts to provide reporting, diagnostics, and campaign warmup tools. The app uses the Meta Marketing API in the following ways:

Data Access

  • GET /me/adaccounts – identifies which ad accounts the user has access to.
  • GET /act_{ad_account}/insights – retrieves basic analytics for display inside the dashboard.
  • GET /act_{ad_account}/campaigns – used only for warmup testing and validation.

Why the Ads API Is Used

The Ads API is used exclusively for:

  • Showing ad account insights to the logged-in user.
  • Confirming correct account permissions.
  • Testing API connectivity through a warmup process.

No Unauthorized Actions

Akonocap Dashboard does not create ads on behalf of a user without consent. Campaign warmup tests only run on reviewer test accounts or on accounts where the user explicitly authorizes the action.
